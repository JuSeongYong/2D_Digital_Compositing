<code>

불필요한 부분은 유리판 위에 매트를 그려 가린 뒤 배우들의 연기를 먼저 촬영하였고

- Digital Matte Painting

<pre>
<code>

컴퓨터의 발전으로 매트페인팅은 두가지의 기술적인 변화가 일어났다.

1. 유화와 붓을 사용한 전통적인 회화기법에서 포토샵과 그래픽 태블릿을 이용한 디지털 드로잉으로 변화한 것이다.

- 매트페인터가 그림을 그리면서 여러 사진을 참조할 수 있음.
- 그림의 수정과 제작이 훨씬 빨라짐; 이전에 그렸던 그림이 기초를 제공해줌.

2. 3D Projection

- 3D환경을 사용해 입체적으로 그림을 투영시킬 수 있다.
- 훨씬 복잡하고 사실적인 표현이 가능해짐.
- 평평한 배경에서 입체적인 환경을 제작할 수 있다.

</code>

- 1920년대 후반에서 1930년대까지 자주 쓰이던 기술.

<pre>

 - 피사체에서 파란색의 흐릿한 가장자리가 나타남.

#### CG(Computer Graphics)

컴퓨터에 의한 영상처리. 미국 매사추세츠공대(MIT) 박사과정을 이수하고 있던 I. 서덜랜드가 1963년에 발표한 「스케치패드 시스템」에 과한 논문에서 시작되었다.
실제로 볼 수 없는 화상을 표시할 수 있다는 이점이 있고 컴퓨터가 출력하는 정보를 시각화하기 때문에 응용범위가 넓다.
최근에는 CG로 약칭되는 경우도 많다. 현재 의약품개발, 지도, 측량, 디자인분야 등 다양한 시스템을 개발, 실용화되고 있다.

- 초기

##### SAGE(Semi-Automatic Ground Environment) : 반자동 방공조직

<pre>
<code>

미국 공군의 NORAD(북미항공우주사령부)에서 1954년부터 가동에 들어간 방공망 관제용의 전산 시스템으로, MIT와 IBM이 중심이 되어 개발된 시스템
최초의 Interactive Computer Graphic을 제공해줬다. 컴퓨터 그래픽의 시작이라고 할 수 있다.

</code>
</pre>

- 1960년대

컴퓨터 그래픽은 1960년대에 실제로 등장했다.
당시 컴퓨터 그래픽에서는 현재 초보적인 조작으로 간주되고 있는 안티앨리어싱 선, 원, 커브 드로잉 등이 주로 있었다.
파라메트릭 곡선과 표면에 대한 Pierre Bézier의 초기 작업이 공개되었고, IBM은 레이 트레이싱의 전구체인 숨겨진 표면과 그림자 알고리즘을 개발했다.
Doug Englebart는 컴퓨터 마우스를 발명했다.

  Bezier Curve

- 1970년대

렌더링(shading)은 유타 대학의 Gouraud와 Phong에 의해 발견되었다.
3D 그래픽용 키프레임 기반 애니메이션을 처음으로 시연했다.
Xerox PARC는 "Paint" 프로그램을 개발했다.
Ed Catmull은 z-buffer 알고리즘을 도입했고 Turned Whitted는 recursive ray tracing을 개발했고, 이는 후에 photorealism의 표준이 되었다.
Pong, Pac-Man과 같은 아케이드 게임은 매우 유명해졌다.

- 1980년대

1980년대 중반에 가까운 시기에 Jim Blinn은 blobby models와 텍스처 맵핑 개념을 도입했다.
Computer Aided Design 강좌를 여러 대학에서 가르치기 시작했다.
데이터 구조로 BSP(Binary Space Partitioning : 이진 공간 분할법) 트리가 도입되었지만, 그것들이 얼마나 유용해질지 깨닫는 사람은 많지 않았다.
몇몇 예술가들이 컴퓨터 그래픽의 프랙탈을 탐구하기 시작했다.
3D 소프트웨어의 목표는 단순한 렌더링이 아닌 캐릭터 애니메이션이 되었다.
어도비는 포토샵 소프트웨어를 시장에 내놓았다.
비디오 게임은 더 나은 컴퓨터 그래픽과 사용하기 쉬운 소프트웨어에 대한 필요성을 만들어 내는 과정에서 시작되었다.

- 1990년대

1992년에 OpenGL은 그래픽 API(Application Program Interface : 운영체제와 응용프로그램 사이의 통신에 사용되는 언어나 메시지 형식)의 표준이 되었다.
충돌·중력·마찰·인과관계로 애니메이션을 가능케 한 역학 시스템이 도입됐다.
최초의 CAD 대학이 개설되었다.
3dfx나 엔비디아와 같은 PC 그래픽 카드가 도입되었다.
데이터 글로브(data glove)에서 시작된 모션 캡쳐는 애니메이션 시퀀스를 생성하는 일차적인 방법이 되었다.
터미네이터 2, 쥬라기 공원, 토이 스토리 같은 영화에 그래픽이 영향을 끼치며 그래픽은 어디에나 존재하기 시작했다.
우리가 21세기로 접어들면서 온라인 CAD 과정을 이용할 수 있게 되었고, 디자인과 기술을 융합하여 건축, 엔지니어링, 그리고 더 많은 분야에서 새로운 변화를 만들었다.
